# Nendoroid List

플러터로 제작된 넨도로이드 정보앱 입니다.

DB제공 : https://github.com/KhoraLee/NendoroidDB

웹버전 : https://silvershort.github.io/web/#/

## 사용된 주요 기술 스택

:bulb: GetX (State Manager)

:bulb: Retrofit (Rest API Call)

:bulb: Hive (Local DB)

:bulb: FCM (Push Messeage)
